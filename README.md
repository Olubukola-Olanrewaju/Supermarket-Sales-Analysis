## Supermarket Sales Analysis Dashboard
## Overview

This project analyzes supermarket sales data to uncover key insights into sales performance, customer behavior, and product trends.

## Tools Used
- Power BI
- Excel

## Dashboard Pages
- Sales Overview
- Product Performance
- Customer Insights
- Branch Performance

## Key Insights
- Branch C generates the highest sales
- Food and Beverages dominate product sales
- Evening hours show peak customer activity
- Members contribute more revenue
